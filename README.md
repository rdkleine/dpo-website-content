# dpo-website-content

Gegenereerde, publiek te hosten content voor de Dutch Pinball Open Expo website.

| Bestand        | Inhoud                                                        |
| -------------- | ------------------------------------------------------------ |
| `program.png`  | De timetable als afbeelding.                                 |
| `program.html` | De timetable als tekstlijst (NL + EN). Bevat een Content-Security-Policy die alle scripts blokkeert. |

Deze bestanden worden **automatisch gegenereerd** door het `dpo-timetable`
project — niet met de hand bewerken. Via GitHub Pages (branch `main`, root) zijn
ze bereikbaar op:

- https://rdkleine.github.io/dpo-website-content/program.png
- https://rdkleine.github.io/dpo-website-content/program.html
